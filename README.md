Liverpool Challenge
===========

### Requerimentos
- Node.js v10.15.1 o posteriores
- MongoDB v4.0x

### Instalar Dependencias

#### yarn
yarn install

#### npm
npm install

## Scripts

Inicializa el cliente de React.js
#### yarn
`yarn start`

#### npm
`npm start`

Inicializa la API
#### yarn
`yarn api`

#### npm
`npm run api`
